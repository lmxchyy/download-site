# ResHub - 多功能资源下载站

基于 PHP + MySQL 的现代化资源下载站，支持资源管理、分类导航、下载统计，Docker 一键部署。

![PHP Version](https://img.shields.io/badge/PHP-8.2-blue)
![MySQL Version](https://img.shields.io/badge/MySQL-8.0-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## 功能特性

- 资源上传/编辑/删除，分类管理
- 下载统计（次数、IP、User-Agent）
- 管理员后台（独立登录认证）
- 响应式设计，适配移动端
- Docker 支持，一键启动

## 快速开始

### Docker 部署


访问 `http://localhost:8080/install.php` 完成安装。

### 手动部署

**环境要求**: PHP 7.4+ / MySQL 5.7+

1. 将 `src/` 目录内容复制到 web 根目录
2. 创建 MySQL 数据库
3. 访问 `install.php` 按向导安装

## 数据库结构

|------|------|
| `categories` | 资源分类 |

## 文件功能详解

| 文件 | 功能 | 亮点 |
|------|------|------|
| `config.php` | 配置、DB连接、辅助函数 | Docker 环境变量优先，自动检测安装状态 |
| `install.php` | 一键安装向导 | 创建5张表、默认分类、admin/admin123 |
| `admin.php` | 管理后台 | 资源 CRUD、分类管理、登录保护 |
| `detail.php` | 资源详情页 | 面包屑、相关资源推荐 |
| `download.php` | 下载处理器 | 记录 IP/UA、分块输出大文件 |

## 配置说明

`config.php` 主要配置项：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| `DB_HOST` | 数据库主机 | `db` (Docker) |
| `DB_USER` | 数据库用户 | `reshub_user` |
| `DB_PASS` | 数据库密码 | `reshub_pass` |
| `DB_NAME` | 数据库名 | `resource_download` |
| `MAX_FILE_SIZE` | 上传文件大小限制 | 50MB |

## 潜在问题与建议

| 问题 | 位置 | 建议 |
|------|------|------|
| SQL 拼接风险 | `admin.php:87,94,105,112` | 改用参数化查询 |
| 上传类型无限制 | `admin.php:54-79` | 添加 MIME 白名单 |
| 缺少 CSRF 防护 | 所有表单 | 添加 CSRF Token |
| 默认密码未强制修改 | `install.php:123` | 添加首次登录强制修改 |
